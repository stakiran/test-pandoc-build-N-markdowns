Scrapbox to Markdown

背景

  • 承認欲求満たしたい
  • ScrapboxだとSEOが弱いし、そもそも承認成分を持ち込みたくない
  • どうするか
    • 静的サイトに変換すればいい
    • 1ページを1ファイルに変換する
  • GitHub Pagesが楽そうでいいかな
    • titやmonolithicみたいなテイストをイメージしている
    • つまりMarkdownに変換する

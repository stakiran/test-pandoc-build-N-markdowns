  • sort by をまるごと消す（他のソートも使えなくなるデメリット）
    • :sta:普段からろくに使ったことないし、これで試してみるか
 /* hidden sort menu especially last visited */
 .page-sort-menu {
  display: none;
 }

/* 同上, last visited item だけ消す */
.page-sort-menu > .dropdown > .dropdown-menu-right > li:nth-child(4) > a:nth-child(1) : {

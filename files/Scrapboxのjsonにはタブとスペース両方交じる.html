Scrapboxのjsonにはタブとスペース両方交じる

  • そのまま挿入されてる
    • タブキーでインデントするとタブ
    • スペースキーでインデントするとスペース
  • ある行から改行して自動インデントした場合、その行のが踏襲される


{
  "name": "sta-export-test",
  "displayName": "sta-export-test",
  "exported": 1609646173,
  "pages": [
    {
      "title": "space-based indent",

      "updated": 1609646164,

      "lines": [
        "space-based indent",

        "   サブ",
        "   \tサブ", //★これはたぶんタブでつくっちまった

    }
  ]
}
